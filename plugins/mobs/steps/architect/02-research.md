# Step 2: Design Research

**Flag**: `-w` or `--websearch`

## Purpose

Research current design trends, patterns, and references using WebSearch MCP to inform the design process.

## What To Do

1. **Generate Search Queries**

Based on the design brief, create targeted searches:

```markdown
## Search Strategy

### Primary Searches
1. "[Project Type] design trends 2025"
2. "[Industry] dashboard/app/website design inspiration"
3. "[Key Feature] UI patterns best practices"

### Style-Specific Searches
4. "Minimalist [Project Type] design examples"
5. "Modern [Project Type] layouts"
6. "[Color Scheme/Style] [Project Type] design"

### Component Searches
7. "[Component Type] design patterns"
8. "[Interaction Type] UX best practices"
```

2. **Execute WebSearch MCP**

Use the WebSearch tool for each query:
- Collect 5-10 high-quality references per search
- Focus on recent (2024-2025) content
- Gather diverse sources (Dribbble, Behance, blogs, case studies)

3. **Analyze and Organize Findings**

Create research summary:

```markdown
# Design Research Summary

## Trends Identified
- [Trend 1]: [Description + Source URL]
- [Trend 2]: [Description + Source URL]
- [Trend 3]: [Description + Source URL]

## Layout Patterns Found
1. [Pattern 1]: [Description + Reference]
2. [Pattern 2]: [Description + Reference]

## Color Palettes
1. [Palette 1]: [Colors + Where seen]
2. [Palette 2]: [Colors + Where seen]

## Typography Trends
- [Trend 1]: [Description]
- [Trend 2]: [Description]

## Component Inspiration
### [Component Type]
- [Example 1]: [URL + Notes]
- [Example 2]: [URL + Notes]

## Key References
1. [URL]: [Why it's relevant]
2. [URL]: [Why it's relevant]
3. [URL]: [Why it's relevant]
```

4. **Save Research**

- Location: `.claude/.smite/design-research.md`
- Include all reference URLs for vision analysis

## MCP Tools Used

- ✅ WebSearch: Multiple queries for trends and patterns
- Optional: WebReader to fetch full content from key URLs

## Output

- ✅ Research summary with 15-30 references
- ✅ Categorized findings (trends, layouts, colors, typography)
- ✅ Reference URLs ready for vision analysis
- ✅ Saved to `.claude/.smite/design-research.md`
