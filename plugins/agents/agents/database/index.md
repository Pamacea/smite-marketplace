# Database Development Agents

Specialized agents for database schema design, migrations, and type-safe data access.

## Available Agents

### Prisma Database Agent
**File**: [prisma.agent.md](./prisma.agent.md)

Specialized in Prisma ORM development with type-safe database operations, migrations, and best practices.

**Best for:**
- TypeScript projects
- Type-safe database access
- Schema migrations
- Database seeding

## Usage

Invoke this agent when:
- Designing database schemas
- Creating migrations
- Setting up ORM
- Writing type-safe queries

## Integration

The Prisma agent works with:
- **backend** agents (NestJS, Rust) for API integration
- **devops** agents (Docker) for local development
