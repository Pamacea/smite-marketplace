# DevOps Development Agents

Specialized agents for containerization, deployment, and infrastructure automation.

## Available Agents

### Docker Development Agent
**File**: [docker.agent.md](./docker.agent.md)

Specialized in containerization with Docker, Docker Compose, and production-ready container orchestration.

**Best for:**
- Containerizing applications
- Local development environments
- Production deployments
- Multi-service applications

## Usage

Invoke this agent when:
- Creating Dockerfiles
- Setting up Docker Compose
- Optimizing container images
- Configuring deployments

## Integration

The Docker agent works with all agents for containerizing their applications.
