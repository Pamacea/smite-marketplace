# Backend Development Agents

Specialized agents for backend development across multiple frameworks and languages.

## Available Agents

### Rust Development Agent
**File**: [rust.agent.md](./rust.agent.md)

Specialized in Rust development with Domain-Driven Design (DDD), async patterns with Tokio, and production-ready best practices.

**Best for:**
- High-performance APIs
- Systems programming
- Microservices
- CLI tools

### NestJS Development Agent
**File**: [nestjs.agent.md](./nestjs.agent.md)

Specialized in NestJS backend development with TypeScript, dependency injection, and modular architecture.

**Best for:**
- Enterprise Node.js applications
- RESTful APIs
- GraphQL backends
- Microservices

## Usage

Invoke these agents when:
- Building backend APIs
- Implementing business logic
- Setting up database schemas
- Creating authentication systems

## Integration

All backend agents use:
- **core** v1.6.5 for shared infrastructure
- **database** agents (Prisma, SQLx) for data access
- **devops** agents (Docker) for deployment
