# Frontend Development Agents

Specialized agents for frontend development with modern frameworks and patterns.

## Available Agents

### Next.js Development Agent
**File**: [nextjs.agent.md](./nextjs.agent.md)

Specialized in Next.js 16 development with React 19.2, Server Components, Turbopack, and production-ready patterns.

**Best for:**
- Full-stack applications
- SaaS platforms
- E-commerce sites
- Content-heavy sites

## Usage

Invoke this agent when:
- Building Next.js applications
- Implementing Server Components
- Creating Server Actions
- Setting up authentication

## Integration

The Next.js agent uses:
- **core** v1.6.5 for templates
- **database** agents (Prisma) for data
- **devops** agents (Docker) for deployment
