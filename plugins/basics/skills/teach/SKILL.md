# Teach Skill - Educational Content Generator

## Mission

Generate educational content from codebase and technical concepts.

## Usage

```bash
/teach "Explain JWT authentication"
/teach --level=junior "How does React Server Components work?"
/teach --diagram "Show payment flow architecture"
/teach --format=html "Create presentation"
```

## Levels

- **Junior** - Simple, analogies, no jargon
- **Mid** - Code examples, patterns, best practices
- **Senior** - Deep dive, trade-offs, performance

## Diagrams

ASCII diagrams for:
- Architecture
- Data flows
- Component trees

## Output Formats

- `markdown` (default)
- `html` (presentation)
- `pdf` (via HTML)
