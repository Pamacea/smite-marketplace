## Overview

The Docs Editor MCP provides three powerful tools:
1. **sync_openapi_spec**: Auto-generate OpenAPI specs from route definitions
2. **update_readme_architecture**: Keep README sections in sync with project structure
3. **generate_jsdoc_on_fly**: Add JSDoc comments automatically

## Installation

```bash
/plugin install docs-editor-mcp@smite
```
