## Overview

Obsidian Note Agent provides:
- Multi-vault support
- Template-based note creation
- Automatic formatting
- Semantic search across vaults
- Intelligent content organization

## Quick Start

```bash
/note --vault="MyVault" --template="meeting"
/note_format --vault="MyVault" "Path/to/note.md"
/search-notes --vaults="MyVault,WorkVault" "query"
```
