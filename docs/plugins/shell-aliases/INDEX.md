## Overview

Shell Aliases provides convenient shortcuts for common SMITE commands in your shell:
- Quick access to SMITE commands
- Persistent aliases across sessions
- Easy installation and configuration

## Quick Start

```bash
/install-aliases
```

Then use shortcuts like:
- `smite-oneshot "add feature"`
- `smite-debug "fix bug"`
- `smite-commit`

See the [main README](../../../plugins/shell-aliases/README.md) for complete documentation.
